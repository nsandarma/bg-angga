SVM

Support Vector Machine (SVM)

Pada dasarnya, support-vector machine merupakan sebuah algoritma klasifikasi untuk data linear dan non-linear. SVM menggunakan mapping non-linear untuk mentransformasikan training data awal ke dimensi yang lebih tinggi.

SVM dapat digunakan untuk memecahkan berbagai masalah dunia nyata:

  • SVM sangat membantu dalam kategorisasi teks dan hiperteks , karena aplikasinya dapat secara signifikan mengurangi kebutuhan untuk instans pelatihan berlabel baik dalam pengaturan induktif dan transduktif standar. Beberapa metode untuk parsing semantik dangkal didasarkan pada mesin vektor dukungan
  • Klasifikasi data satelit seperti data SAR menggunakan SVM terawasi
  • Karakter tulisan tangan dapat dikenali menggunakan SVM
